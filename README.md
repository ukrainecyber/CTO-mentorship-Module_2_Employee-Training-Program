# Module 2 Establishing and implementing a cybersecurity employee training & testing program 

You can force complex passwords, install drive encryption, and use three different types of security applications, yet, none of that matters unless you tackle the primary cause of security issues: your employees. While typically not a malicious act, employee caused security incidents are quite common, with phishing attacks accounting for 43 percent of all data breaches. Besides technical means, a comprehensive Security Awareness Training program is essential when it comes to your overall security plan.
